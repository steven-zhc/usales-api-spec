purchase and sell request.

/order get post

get

Get a list of active order.

post

Create a new order

get /order

Get a list of active order.

  • Request
  • Response

Query Parameters

  • name: (string - minLength: 3)

    The name of product to search (it is case insensitive and doesn't need to match the whole name)

    Examples:

    blue
  • status: (one of 0, 1, 2, 3, 4)

    The category ID

    Examples:

    1

HTTP status code 200

Body

Media type: application/json

Type: array of Order Type

Items: Order Type

  • oid: required (integer)
  • status: required (one of 0, 1, 2, 3, 4)
  • total: required (number)
  • profit: required (number)
  • date: required (datetime)
  • note: (string)
  • deliverFee: (number)
  • deliverDate: (date-only)
  • trackingNo: (string)
  • payment: (number)
  • lines: required (array of Order Line)

    Items: Order Line

    • lid: required (integer)
    • pid: required (integer)
    • date: required (datetime)
    • total: (number)
    • profit: (number)
    • model: (string)
    • note: (string)
    • q: required (integer)
    • purchase: required (object)
      • p: required (number)
      • t: required (number)
      • s: required (number)
      • d: (number - maximum: 0)
      • total: (number)
    • sell: required (object)
      • p: required (number)
      • t: required (number)
      • s: required (number)
      • d: required (number - maximum: 0)
      • total: required (number)

    Examples:

    {
  "lid": 1,
  "pid": 1,
  "date": "2016-03-01T14:30:03Z",
  "total": 23.67,
  "profit": 3.99,
  "model": "SL33",
  "note": "A gift for brother",
  "q": 2,
  "purchase": {
    "p": 10.0, "t": 0.88, "s": 0.00, "d": -1.20, "total": 19.68
  },
  "sell": {
    "p": 10.0, "t": 0.88, "s": 3.99, "d": -1.20, "total": 23.67
  }
}

Examples:

{
  "oid": 1,
  "status": 2,
  "total": 23.67,
  "profit": 3.99,
  "date": "2016-03-01T14:30:03Z",
  "lines": [
    {
      "lid": 1,
